# 🚨 Phishing Email Analysis Project

## 🔍 Overview
This project analyzes suspicious emails to detect phishing attempts using Python and Jupyter Notebook.

It simulates a real-world **SOC Analyst workflow** by identifying malicious indicators like suspicious links and urgency-based language.

---

## 🛠 Tools Used
- Python 🐍
- Jupyter Notebook 📓
- Regex 🔍
- Pandas 📊

---

## ⚙️ Analysis Approach
1. Extract URLs from email
2. Detect phishing keywords (urgent, verify, click, etc.)
3. Assign risk score
4. Classify email as **Phishing or Safe**

---

## 📊 Output Preview

| URLs Found | Keywords Found | Risk Score | Result |
|------------|---------------|------------|--------|
| 1          | 5             | 4          | PHISHING |

## 📁 Project Files
